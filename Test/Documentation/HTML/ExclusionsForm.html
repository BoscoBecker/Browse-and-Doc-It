 TfrmExclusions

Name Data / Type
 Execute 
Procedure Execute(
  slExclusions : TStringList
)

This is the forms main interface method for invoking the edit dialogue .

Pre-Conditions

  • slExclusions must be a valid instance of a string list .

Post-Conditions

  • If the dialogue is confirmed the passed string list is updated with the edited list of exclusions .

Parameters

  • slExclusions as a TStringList
