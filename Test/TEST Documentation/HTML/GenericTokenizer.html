Function Tokenize(
  strText : String;
  var KeyWords : TKeyWords;
  iLimit : Integer = 999999
) : TStringList

This function returns a string list contains the tokenized representation of the passed string with respect to some basic object pascal grammer .

Pre-Conditions

  • strText si the line of text to be tokenised

Post-Conditions

  • Returns a new string list of the tokenized string

Parameters

  • strText as a String
  • KeyWords as a TKeyWords as a reference
  • iLimit as an Integer

Returns

  • a TStringList

Notes

  • The string list returnsed must be destroyed be the calling method .
