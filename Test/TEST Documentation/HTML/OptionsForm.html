 Execute 
Function Execute : Boolean

This method creates an instance of the options dialogue and sets all the controls based on the passed Options parameter . If OK is selected then the Options parameter is updated to suit the new options .

Pre-Conditions

  • iInt is the timer interval to be represented in the dialogue , DocHelpFile is the directory of the modules help file .

Post-Conditions

  • Returns true if the OK button on the dialogue was pressed .

Returns

  • a Boolean
