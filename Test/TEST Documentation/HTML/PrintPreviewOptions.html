Function Execute(
  Preview : TSynEditPrintPreview
) : Boolean

This the method used to invoke the preview option dialogue .

Pre-Conditions

  • Preview . SynEditPrint must be a valid TSynEditPrint class .

Post-Conditions

  • If changes are made and the OK button is pressed then the Preview . SynEditPrint class is updated with the changes .

Parameters

  • Preview as a TSynEditPrintPreview

Returns

  • a Boolean

Notes

  • Note that in this implementation the Footers and Headers of the Preview . SynEditPrint class are edited directly .

 PopulateListBox 
Procedure PopulateListBox(
  ListBox : TListBox;
  HeaderFooter : THeaderFooter
)

This method populate the given list box with the text of the given header or footer so that they have the same number of items and list order .

Pre-Conditions

  • Both Listbox and HeaderFooter must be valid instances of a TListBox and THeaderFooter classes respectively .

Post-Conditions

  • Populate the given list box with the text of the given header or footer .

Parameters

  • ListBox as a TListBox
  • HeaderFooter as a THeaderFooter
