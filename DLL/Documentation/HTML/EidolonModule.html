 Documentation for Unit EidolonModule

EidolonModule : A unit to parser Eidolon code . Please refer to the file "Eidolon Map File Grammar.bnf" for the complete grammar implemented .

Date Code Last Updated

  • 13 Nov 2012

Code Author

  • David Hoyle

Code Version

  • 1 . 0

 Notes

  • Not currently configured or used.
  • All comments found are automatically added to the comment collection for disposal when the parser is destroyed.

 Debugging Code

  • This does not seem to cause an error here!!!!
