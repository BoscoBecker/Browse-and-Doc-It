 lvListView1CustomDrawItem 
Procedure lvListView1CustomDrawItem(
  Sender : TCustomListView;
  Item : TListItem;
  State : TCustomDrawState;
  var DefaultDraw : Boolean
)

This is a custom draw method for the TListView component . It colours the items differently depending on the contents of the item . caption property .

Pre-Conditions

  • Sender is the object initiating the event , Item is the item in the list view being displayed , State is the state of the item being displayed and DefaultDraw can be changed to to define default drawing or owner drawing .

Post-Conditions

  • This method colours the items in the list view based on their type .

Parameters

  • Sender as a TCustomListView
  • Item as a TListItem
  • State as a TCustomDrawState
  • DefaultDraw as a Boolean as a reference
