 Documentation for Unit PascalModule

ObjectPascalModule : A unit to tokenize Pascal source code .

Things To Do

  • Implement an expression parser for the above compiler defines . Needs to be able to evaluate constants in the code and use the two functions Defined () and Declared ().

Date Code Last Updated

  • 20 Apr 2013

Code Author

  • David Hoyle

Code Version

  • 1 . 0

 Things To Do

  • Implement an expression parser for the above compiler defines. Needs to be able to evaluate constants in the code and use the two functions Defined() and Declared().

 Notes

  • All comments found are automatically added to the comment collection for disposal when the parser is destroyed.
  • Does not get added to any symbols.
  • This may have problems with expression that allow integers and floats to be added, etc.

 Also See

  • For object pascal grammar see {@link TPascalDocModule.VarSection} .
