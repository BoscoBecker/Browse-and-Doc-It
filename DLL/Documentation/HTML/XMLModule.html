 Documentation for Unit XMLModule

XMLModule : A unit to parser XML . Please refer to the file "Extensible Markup Language (XML) 1.0.bnf" for the complete grammar implemented .

Date Code Last Updated

  • 26 Jun 2010

Code Author

  • David Hoyle

Code Version

  • 1 . 0

 Things To Do

  • Should be either %Name; or &#38;#38;Name; or &#38;#38; Number; or &#38;#38; xHexNumber;

 Notes

  • Not currently configured or used.
  • All comments found are automatically added to the comment collection for disposal when the parser is destroyed.
